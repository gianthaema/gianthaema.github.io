#### 2. 구글 애널리틱스 적용

1. 구글 애널리틱스 가입

2. 블로그 등 정보 입력

3. 발급된 추적ID를 _config.yml에 등록

   ```
   # Analytics
   analytics:
     provider               : "google-gtag" # 구글 애널리틱스 등록
     google:
       tracking_id          : "UA-134677596-2" # 추적ID
       anonymize_ip         : # true, false (default)
   ```
