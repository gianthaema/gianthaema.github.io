## __자이언트해마의 기술블로그 시작__

## __1. GitHub 블로그 만들기__

1. 저장소 생성
2. Jekyll 테마 다운로드
3. _config.yml 설정 파일에 url, title 등 수정
4. _posts/ 경로에 새 글 작성
5. git에 push

## __2. 구글 애널리틱스 적용__
1. 구글 애널리틱스 가입
2. 블로그 등 정보 입력
3. 발급된 추적ID를 _config.yml에 등록

```
#발급된 추적 ID 등록
google_analytics:   UA-148800000-1 
```
